
# PalindromeCheckerCPP

A simple C++ program that reverses an integer and checks if it is a palindrome.

## 📌 Features
- Takes an integer input from the user.
- Reverses the digits of the number.
- Compares the reversed number with the original.
- Prints whether the number is a palindrome or not.
